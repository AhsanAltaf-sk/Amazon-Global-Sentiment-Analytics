Amazon Product Sentiment Analysis Dashboard
📌 Project Overview
This repository contains a comprehensive Sentiment Analysis Dashboard built in Power BI. The project focuses on analyzing customer feedback from Amazon reviews to understand public opinion, regional trends, and sentiment shifts over time.

📊 Key Features
Sentiment Distribution: A high-level overview of Positive, Negative, and Neutral feedback.

Time-Series Analysis: A line chart depicting sentiment trends from 2007 to 2024.

Geographical Breakdown: Stacked bar charts showing sentiment ratios by country (US, GB, FR, IN, etc.).

Dynamic Filtering: Integrated Date Slicers for granular time-period analysis.

🛠 Tools & Technologies
Tool: Power BI Desktop

Data Processing: Power Query (ETL)

Visuals: Cluster Column Charts, Line Charts, Stacked Bar Charts, Donut Charts.

Techniques: Text Classification, Exploratory Data Analysis (EDA).

💡 Business Insights (Task 5)
Identified that while 48% of reviews are positive, a significant 43% negative sentiment requires immediate attention in specific regions like the US and UK.

Observed a massive scale-up in review volume post-2015, indicating rapid market expansion.

Recommended targeted marketing campaigns for the French market, which shows the highest positive sentiment (58%).
